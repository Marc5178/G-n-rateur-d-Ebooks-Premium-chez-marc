# 📚 Générateur d'Ebooks Automatique chez marc

Un site web moderne qui permet de créer des ebooks automatiquement simplement en ajoutant un titre.

## 🚀 Fonctionnalités

### ✨ Caractéristiques principales
- **Génération automatique** : Créez des ebooks complets avec seulement un titre
- **Personnalisation avancée** : Choisissez le genre, la longueur et le ton
- **Multiple formats** : Téléchargez vos ebooks en PDF, EPUB ou TXT
- **Interface moderne** : Design responsive et élégant
- **Aperçu en temps réel** : Visualisez votre ebook avant de le télécharger

### 📖 Genres disponibles
- Fiction
- Non-fiction
- Science-fiction
- Fantasy
- Romance
- Thriller
- Biographie
- Développement personnel

### 📏 Options de longueur
- **Court** : 1000-2000 mots (3 chapitres)
- **Moyen** : 3000-5000 mots (5 chapitres)
- **Long** : 6000-10000 mots (8 chapitres)

### 🎭 Tons disponibles
- Formel
- Décontracté
- Humoristique
- Dramatique
- Inspirant

## 🛠️ Installation

1. Clonez ou téléchargez ce repository
2. Ouvrez le fichier `index.html` dans votre navigateur web
3. C'est tout ! Aucune installation requise

## 📱 Utilisation

### Étape 1 : Saisir le titre
- Entrez le titre de votre ebook dans le champ prévu à cet effet
- Le compteur de caractères vous aide à respecter la limite de 100 caractères

### Étape 2 : Personnaliser
- **Genre** : Sélectionnez le genre littéraire de votre choix
- **Longueur** : Choisissez la longueur souhaitée pour votre ebook
- **Ton** : Définissez le ton d'écriture

### Étape 3 : Générer
- Cliquez sur le bouton "Générer l'ebook"
- Attendez quelques secondes pendant que le contenu est créé

### Étape 4 : Télécharger
- Visualisez l'aperçu de votre ebook
- Choisissez le format de téléchargement :
  - 📄 **PDF** : Pour une impression ou lecture sur ordinateur
  - 📱 **EPUB** : Pour les liseuses et mobiles
  - 📝 **TXT** : Format texte brut universel

## 🔄 Comment ça fonctionne

1. **Analyse du titre** : Le système analyse le titre pour comprendre le thème
2. **Génération de structure** : Création automatique des chapitres selon le genre
3. **Rédaction du contenu** : Génération de texte cohérent avec le ton choisi
4. **Formatage** : Mise en forme selon le format de téléchargement

## 📊 Statistiques générées

Pour chaque ebook, le système affiche :
- **Nombre de mots** : Comptage précis du contenu
- **Nombre de pages** : Estimation basée sur 250 mots/page
- **Nombre de chapitres** : Selon la longueur choisie
